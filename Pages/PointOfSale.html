Point of Sale System using Visual Basic

a strong system that creates analytical reports to help your business grow while handling personnel, suppliers, expenses, and product inventory with simplicity.

Dashboard

A user-friendly interface that is easy to understand. The dashboard contains a side menu, displays important data (like today's sales, stock, today's purchases, today's expenses, and expired products), and also shows recent invoices.

Side menu options

  • New Sale
  • Products
  • Employees
  • Supplier
  • Reports
  • Notification
  • Settings

It automatically calculates today's sales and displays them on the dashboard. It also calculates stock and products. Additionally, it identifies expired products to replace or reorder them.

New Sale

This window is crucial because it is where we create receipts for customers. Each product has a unique barcode, so when we scan the barcode, it displays the product's data (such as price, quantity, category, and more). The system automatically generates the receipt for the customer and displays it in the same window.

At the bottom of the window, there are fields for discount and cash. The total number of quantities, total amount, and return balance are also displayed.

There is a button in the bottom right corner for saving and printing. If we click on it, this will print the receipt and also save the receipt's data and details in the background.

We can view all the receipt data by clicking on the 'View Reports' button.

Invoice Records

After pressing 'View Report' in the New Sale, a window containing all of the receipt data and search fields appears. With every row, there are two buttons that we can use to examine or reproduce any prior receipt.

Add and View Product

Here, we fill the form below to add a new product to our database. A date selection, drop-down menus, and several fields are present. We add items to the drop-down menus based on our needs. User-Article, Barcode, and Product-Id are the three parameters for product identity.

We can see the product details in the window above after adding a new product. To search for a product, use the four fields (Product name, Barcode, Category, and Price). A remove button is also available for taking out any product. To update any product, we can use the Alt+E shortcut.

Add and View Employee

We enter the employee's information and picture into the database in this window. Fields, drop-down menus, date selectors, and buttons are all available. We add things to the drop-down menu based on our needs.

We can see all employee information in this window. If we want to update any product, we can do so by pressing Alt+E. We can also remove any product by clicking on the red button next to each row.

Add and View Supplier

If you want to keep information about suppliers, you can enter all the information about the supplier and save it into the database. There are also some fields, a drop-down, and a button. Additionally, there is a picture box for inserting a picture.

Here, we see all the information about the suppliers. If we want to see specific supplier information, we can do so by using the search boxes. Additionally, if we want to update it, we can do this by pressing Alt+E, and to remove it, we have to press the remove button next to each row.

Add and View Expense

If we want to keep information about expenses, we have to add each expense using this window. There are some fields, a drop-down, and a date selector.

After filling in the expense information and clicking on 'Add to List,' all data from the fields will be added to the grid view. Once we have entered all expenses, we can save them by clicking on the 'Save All' button.

The above window shows the details of expenses and includes some fields for searching. If we want to remove or edit any expenses, we can do so by clicking on the button next to each row.

Notification Center

This window is for managing notifications. It shows all expired, low stock, and out of stock products. There are three tabs which display expired, low stock, and out of stock products separately. There are also some fields for searching.

Reports

This window is for generating reports between two dates. When we select two dates and click on the 'Generate report' button, it will generate the report of total sales, cost price, and total expenses.

At the bottom of the window, the total number of products, total sale amounts, cost amounts, total expenses, and the calculated profit are displayed.

The above picture shows the report in a graphical view.

Manage Passwords and Users

This window provides user and password management. We can easily change the admin password and name. We can also add new users or view all users in this view.

Settings

Here are some settings, such as changing the company logo. There are also some settings for the receipt, like changing the company name, company logo, contact number, and address. Additionally, there are two checkboxes for showing the logo on the receipt and the same for the company name.
